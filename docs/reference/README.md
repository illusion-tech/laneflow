# 参考资料

本目录用于保存 LaneFlow 的长期参考资料、模板、术语和通用约定。

`docs/reference/` 不承载高层决策，也不替代设计文档。它用于沉淀可复用的辅助规范。

## 适合放在这里的内容

- 术语表
- 状态字典
- 命名约定
- Issue 和 PR 模板说明
- 校验矩阵
- 数据格式版本约定
- 示例场景验收清单

## 当前文档

- `commit-convention.md`：提交信息规范，用于记录 Issue、Gate、切片类型、影响范围、验证结果和文档回写状态。
- `validation-matrix.md`：切片类型到最小验证要求的矩阵，用于 `G3` 合并闸口判断。
